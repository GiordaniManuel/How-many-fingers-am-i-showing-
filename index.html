Project 1:

BBC Homepage Clone:

A clone of a BBC page built using HTML and CSS. The main goal was to replicate the layout and style of the original site, focusing on the page structure, multimedia content, and responsive design.

Project 2:

Three Circles Challenge:

This project features a website centered around three interactive circles.The objective was to make each circle disappear upon clicking, utilizing JavaScript for the functionality. The site also incorporates HTML and CSS to create a visually appealing layout and style. This project highlights my skills in JavaScript, demonstrating how to manipulate the DOM and create engaging user experiences.

Project 3:

Guess the Number of Fingers Game:

In this interactive game, the user is presented with a random number of fingers shown, and their task is to guess the number by typing their answer into an input field. The game is primarily focused on JavaScript, which generates the random number and checks the user's guess. HTML and CSS are used to create a simple layout
